# API Backend cho Quản Lý Sản Phẩm

## Mô Tả

API này được xây dựng để quản lý thông tin sản phẩm. Dự án sử dụng Node.js với Sequelize để tương tác với cơ sở dữ liệu.

## Công Nghệ Sử Dụng

- Node.js
- Express
- Sequelize
- MySQL
